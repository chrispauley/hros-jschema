About

This is an independently developed web application to facilatate the development of the HROS Standards. The index.html page hosts the HROS approved schema validation tool, JSchema. This will help the average person to work with the schemas without the required tooling.

Project Design

The project uses NPM and Bower to pull the repositiories into the public folder. The contents of the public folder is served to the web. This includes an the JSchema web application which is embedded into the index.html page with an iframe tag. Simple, but effective.

The JSchema web application is served in an iframe tag. Links on that page load the workgroup schemas into the web application.

The whole project is wrapped in a Docker image with a small linux instance and an Nginx web server. The docker image is tagged and pushed to a Docker Hub account.

To run a the docker image, one can pull a copy of the image and run it as a container in an environment such as AWS EC2. One could run the container on their local machine or anywhere that has Docker installed.
